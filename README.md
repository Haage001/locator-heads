- Enable/Disable Mod

- Render Heads
  Choose if you want heads to be rendered (can be used to only show names)
- Head Size

- Show Player Names

## 游꿢 Advanced Player Filtering

choose who appear on your locator bar:

- **All Players** _(default)_  
  Shows everyone.
- **Include Only**  
  Display only the players listed in your filter.
- **Exclude Only**  
  Hide only the players listed, showing all others.

**Filter Syntax**  
- Separate names by commas (`,`), semicolons (`;`), colons (`:`), or periods (`.`).  

> **Example:**  
> ``Jeb_, Dinnerbone; LadyAgnes``
